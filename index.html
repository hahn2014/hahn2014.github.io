About Mift

Mift is a word Mason Cluff and I invented while coming up with a name for the game. It is a mix of the Myo and the Oculus Rift. Nearing the end of my Junior year, I came up with the idea to develop a 3D game that implements the Oculus Rift and MYO Armband. I threw a concept idea to our teacher, explaining how I would build a 3D engine and develop the game from the ground up then implement these features. The idea intrigued Mr. Galbraith, leading to the purchase of the Oculus Rift DK2. I worked on Mift all throughout the summer, pushing through the bulk of the 3D engine, creating a playable(ish) game to preview at the beginning of my senior year. Nearing the end of the year, many large features have been implemented, too many to list (I recommend checking out the Mift Repository). Although the Oculus Rift has not yet been implemented into the game, it is a feature the Mift team definitely plans to add later on down the road. Our goal is to successfully have implemented at least the Myo Armband into the game prior to OGPC 2016 (currently under implementation stages). Mift is a simple 3D game project which is not to be taken seriously as for it is still in pre-alpha stage. Post high school development plans are still unknown as a team, but I plan to continue working on Mift until release!

-Bryce Hahn

How To Play

  • Download Desired Executable ZIP from downloads page
  • Unzip ALL files and folders from zip to desired directory.
  • Change settings from Mift.set file as you please
  • Run downloaded Mift file:
    • Mift-windowsX64X32.exe for Windows 7 and above
    • Mift-macosxX64X32.app for Mac OS X Snow Leapord and above
    • Mift-crossplatX64X32.jar for any platform with Java Runtime Environment 1.8 or above
  • Do not delete or move any folders and subfiles! The game requires them to run and will no longer run.
